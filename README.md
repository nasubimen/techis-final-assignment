# 商品在庫管理システム

## 概要

このシステムは商品の在庫を管理することができます。
商品の登録、編集、削除を行うことができ、各商品の在庫を調整することが可能です。

## 主な機能

-   ログイン・ログアウト機能
-   商品一覧画面
-   商品新規登録・編集・削除機能
-   商品検索機能
-   カテゴリー別管理
-   在庫管理履歴機能
-   ログイン・ログアウト機能

## 開発環境

PHP 8.1.12
MySQL 5.7.34
Laravel 8.83.3
